NAME
----
ruby193-rake - wrapper for running rake inside of ruby193 Software Collection


SYNOPSIS
--------
ruby193-rake <params> ...

DESCRIPTION
-----------

ruby193-rake is simply wrapper for
  scl enable ruby193 "rake <params> ..."
